# Power BI Dashboard

Use `data/cleaned_supply_chain_data.csv` to build dashboard pages.

## Suggested Dashboard Pages

1. Executive Summary
2. Supplier Risk Scorecard
3. Procurement Spend Analysis
4. Inventory Stockout Risk
5. Delivery Performance
6. Regional / Category Insights

## Suggested KPIs

- Total PO Spend
- Total Purchase Orders
- Late Delivery Rate
- Defect Rate
- Average Lead Time
- High-Risk Suppliers
- Products Below Reorder Point
