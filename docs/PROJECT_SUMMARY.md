# Project Summary

## Project Name

AI Supply Chain Control Tower Agent

## Business Problem

Supply chain teams often work across fragmented spreadsheets, ERP extracts, supplier scorecards, and inventory reports. This makes it difficult to quickly identify supplier risk, stockout exposure, procurement cost drivers, and operational delays.

## Solution

This project creates a GenAI-enabled supply chain control tower that allows business users to ask natural-language questions and receive KPI-driven answers from supply chain data.

## Key Capabilities

- Supplier risk scoring
- Late delivery analysis
- Inventory stockout risk detection
- Procurement spend analysis
- Executive summary generation
- Supplier email drafting
- PostgreSQL-ready data model
- Streamlit chatbot interface

## Tools Used

- GPT-5.5 / OpenAI API
- Python
- Pandas
- PostgreSQL
- SQL
- Streamlit
- Power BI-ready CSV dataset

## Resume Bullet

Developed an AI-powered Supply Chain Control Tower Agent using Python, PostgreSQL, SQL, Streamlit, Power BI-ready datasets, and GPT-5.5 to analyze supplier risk, procurement spend, late deliveries, inventory exposure, and stockout risk through natural-language business queries.
